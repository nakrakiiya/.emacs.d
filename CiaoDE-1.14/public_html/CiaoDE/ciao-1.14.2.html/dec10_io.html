DEC-10 Prolog file IO

This module implements the support for DEC-10 Prolog style file I/O.

Usage and interface

  • Library usage:
    :- use_module(library(dec10_io)).
  • Exports:
    • Predicates:
      see/1, seeing/1, seen/0, tell/1, telling/1, told/0, close_file/1.
