Documentation on exports

PREDICATE
img_url/2:

Usage: img_url(Name,Url)

  • Description: Obtain the URL where image Name is or will be found.
  • Call and exit should be compatible with:
    (atm/1)Name is an atom.
    (string/1)Url is a string (a list of character codes).
